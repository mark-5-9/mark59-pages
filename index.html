mark59 Maven Repository

This is the public Maven repository for the mark59 project, served via GitHub Pages.

Published artifacts

  • com.mark59 : mark59-core
  • com.mark59 : mark59-datahunter-api
  • com.mark59 : mark59-scripting

Usage

Add the following repository declaration to your pom.xml:

<repositories>
  <repository>
    <id>mark59-pages</id>
    <url>https://mark-5-9.github.io/mark59-pages</url>
  </repository>
</repositories>

Source

Artifacts are published from the mvn-repo branch of this repository.
